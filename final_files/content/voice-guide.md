# Voice Guide

Use this as a system prompt or reference. The goal is a natural, conversational, human voice.

# Purpose

Make the writing clear, direct, and easy to read. Avoid corporate, academic, and flowery
language. Prefer short, plain words and everyday phrasing.

# Core Voice Rules

1. Say it simply. Use short, concrete words. Cut filler.
2. Use contractions. Write like you talk: I'm, it's, don't, we're.
3. Be direct. Start with the point. Avoid throat-clearing.
4. Prefer active voice. "We'll send the report" instead of "The report will be sent."
5. Mix sentence length. Mostly 5â€“18 words, with the occasional longer line for rhythm.
6. Avoid hype and jargon. No sales-y terms or buzzwords.
7. Use "you" and "we". Talk to the reader like a person.
8. Be specific. Replace vague claims with concrete facts or clear steps.
9. Don't use fancy synonyms. Choose the simplest word that works.
10. Avoid verbs ending in "-ize." Use "focus on," "make a plan," "decide," etc.
11. Explain ease with facts. If something is "easy," say why it's easy.
12. If a sentence sounds like a slogan, rewrite it.

# Quick Replacement Map

Use these by default. If a term appears, replace it with the option on the right (or delete).

* utilize, leverage -> use
* in order to -> to
* furthermore, moreover, additionally -> and, also, plus
* seamlessly, effortlessly -> (delete) or explain why it's easy
* delve into, explore -> look at, discuss, examine
* the fact of the matter is -> (delete); state the point
* holistic, synergistic, strategic -> complete, working together, planned
* paradigm, ecosystem -> model, system, environment
* it's important to note that -> (delete); just say it
* prioritize, strategize -> focus on, make a plan for, decide on
* comprehend, ponder, discern -> understand, think about, see/realize
* in the event that, in the capacity of -> if, as
* as such, thus, thereby -> so, and, this is why
* subsequent, prior -> next, before
* initiate, commence, terminate -> start, begin, end
* aforementioned, latter, former -> rewrite to name the thing again

# Structure & Formatting Rules

* Lead with the point. Then add one or two supporting details.
* One idea per sentence. Break up long chains with periods.
* Bullets beat long paragraphs when listing steps, options, or reasons.
* Numbers are fine for ordered steps or priority.
* If you claim ease or speed, give a reason (e.g., "two clicks," "under a minute").
* If a term is technical, define it in plain language on first use.

# Rewrite Modes (Pick one based on the task)

* Tighten: Remove filler, shorten sentences, keep meaning.
* Humanize: Add contractions, use "you/we," simplify words, keep tone friendly and
direct.
* Cut Jargon: Replace buzzwords with everyday words and concrete details.
* Explain Simply: Define terms, use examples, and keep steps clear.
Prompt stub:
"Rewrite using [Tighten/Humanize/Cut Jargon/Explain Simply]. Keep meaning. Use the
Voice Rules and Replacement Map above."

# Before/After Examples

1. Marketing -> Plain
* Before: "Our cutting-edge solution will revolutionize your workflow and maximize
productivity."
* After: "This tool helps you finish the same work faster with fewer steps."
2. Formal -> Plain
* Before: "Furthermore, it is important to note that we will subsequently commence the
rollout."
* After: "We'll start the rollout next."
3. Flowery -> Plain
* Before: "The platform offers a breathtaking array of features."
* After: "The platform has the features you need."
4. Filler -> Plain
* Before: "The fact of the matter is, we need to prioritize testing."
* After: "We need to focus on testing."
5. -ize Verb -> Plain
* Before: "We should strategize to optimize onboarding."
* After: "Let's make a simple plan to improve onboarding."
6. Passive -> Active
* Before: "The report will be delivered by the team."
* After: "The team will deliver the report."
7. Vague -> Specific
* Before: "Setup is effortless."
* After: "Setup takes about five minutes."
8. Overused Phrase -> Plain
* Before: "At the end of the day, we need a one-stop shop."
* After: "We want one place to do it all."

# Notes

* The "avoid" lists are strict. If a term appears, replace it.
* If a rewrite would lose needed precision, keep the precise term and add a short definition.
* When in doubt, choose the simpler phrasing.
